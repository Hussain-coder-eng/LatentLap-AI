## Codebase Patterns
- TireParticles: base circles at cx=200 cy=200; use translateX/translateY keyframes (not cx/cy attrs) to orbit — more reliable with animejs v4
- animejs v4 `.pause()` is the cleanup method for looping animations in useEffect return
- globals.css lives in dashboard/styles/globals.css (not dashboard/app/globals.css)
- RaceContext.tsx uses useMemo for the context value — add new state vars to both the state block and the useMemo deps array
- animejs v4 uses named exports: `import { animate, stagger, morphTo, createDrawable, createMotionPath, utils } from 'animejs'` — NOT the default export `anime({...})`
- GSAP ScrollTrigger must be registered: `gsap.registerPlugin(ScrollTrigger)` before use
- TypeScript check: `cd dashboard && npx tsc --noEmit`
- Dashboard Python env: `~/.venv/bin/python`
- All new dashboard components use NAMED exports (not default) — SilverstoneCircuit, CalloutLeft, CalloutRight, TireRings, TireTread, TireParticles. TireHero uses default export.
- JSON import path from dashboard/app/components/: `../../public/data/filename.json` (2 levels up = dashboard root, then public/data)
- `string(x)` is invalid TS — use `String(x)` or `` `${x}` `` for runtime type coercion
